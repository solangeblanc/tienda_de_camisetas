royecto: E-commerce de Indumentaria Deportiva (Camisetas)

Este proyecto, desarrollado para la cátedra de Diseño y Desarrollo Gráfico, consiste en un MVP (Mínimo Producto Viable) de una tienda online de camisetas. El objetivo principal fue trabajar la experiencia de usuario (UX) y la interfaz (UI) desde una arquitectura escalable. Aunque el sitio se presenta en una fase inicial, su estructura técnica y visual está diseñada para integrar futuras funcionalidades, demostrando la capacidad de crecimiento y adaptación del producto a un mercado real.
